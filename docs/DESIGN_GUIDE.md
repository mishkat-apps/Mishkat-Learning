# Mishkat Learning - Design Guide

This document outlines the design system and UI structure for the Mishkat Learning app, based on analyzed mockups.

## Color Palette
- **Primary (Deep Emerald)**: `#0E6B5B`
- **Secondary (Midnight Navy)**: `#0B1324`
- **Accent (Warm Gold)**: `#C8A24A`
- **Surface (Soft Sand)**: `#F4F0E6`
- **Text**: `#111827`

## Typography
- **Font**: Roboto or Outfit (Sans-serif)
- **Scale**:
  - H1: 32px Bold
  - H2: 24px Bold
  - Body: 16px Regular
  - Small: 14px Regular

## Components
- **Cards**: 16px border-radius, soft shadows.
- **Buttons**: 12px border-radius, Primary or Outlined style.
- **Progress Bars**: Emerald background with Gold fill.
- **Navigation**: Bottom nav for mobile, Sidebar for desktop.

## Breakpoints
- Mobile: < 600px
- Tablet: 600px - 1024px
- Desktop: > 1024px
